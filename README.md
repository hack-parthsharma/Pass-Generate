# PassGenerate
A program that generates random safe password. (Python 3)

[![Python: 3.5](https://img.shields.io/badge/python-3.5-blue)](https://www.python.org/downloads/release/python-350/)

### Installation

Shell:

```shell
$ ./install.sh
```
### Usage

```shell
$ PassGenerate [sizeOfPassword]
```
